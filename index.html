BitsyBASIC is an implementation of Tiny BASIC for iPhone, iPad, and iPod touch devices running iOS 8 or later.

It is useful as

  • a simple introduction to computer programming
  • a programmable calculator
  • a simulation of a mid-1970's personal computer for those who want to relive the good old days
  • a puzzle for those who want to face the challenge of producing useful programs with a minimal programming language and development environment

See the tutorial for a guide to getting started with using BitsyBASIC and writing simple programs.

See the reference for details about the syntax and behavior of the BitsyBASIC language.
